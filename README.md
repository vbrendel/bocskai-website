# Bocskai Hungarian School Website

A Jekyll website for the [Bocskai Hungarian School](https://www.bocskaihungarianschool.com.au/) — an accredited community language school in Melbourne, Australia.

## Requirements

- Ruby 3.x
- Bundler (`gem install bundler`)

## Getting Started

```bash
# Install dependencies
bundle install

# Serve locally with live reload
bundle exec jekyll serve --livereload

# Build for production
bundle exec jekyll build
```

The site will be available at `http://localhost:4000`.

## Structure

```
.
├── _config.yml          # Jekyll configuration
├── _layouts/
│   ├── default.html     # Base layout (html, head, header, footer)
│   └── page.html        # Inner page layout (adds page-hero + breadcrumb)
├── _includes/
│   ├── head.html        # <head> tag contents
│   ├── header.html      # Site header + navigation
│   └── footer.html      # Site footer
├── assets/
│   ├── css/main.scss    # All styles (compiled by Jekyll)
│   └── js/main.js       # Nav toggle + form handling
├── index.html           # Home page
├── about-us.md          # About Us page
├── enrolments.md        # Enrolments page
├── community.md         # Community page
├── contact-us.md        # Contact Us page
```

## Pages

| Page | URL | Description |
|------|-----|-------------|
| Home | `/` | Hero, class overview, about, stats, library CTA |
| Enrolments | `/enrolments/` | Enrolment steps, schedule, fees |
| Community | `/community/` | Facebook, volunteering, wider community |
| School Life | `/school-life/` | Events, gallery, school year calendar |

The Library links externally to the URL set in `library_url` in `_config.yml`.

### Colours & Fonts
All design tokens are CSS custom properties at the top of `assets/css/main.scss`:
- `--color-primary` — navy blue (`#2c5f8a`)
- `--color-secondary` — Hungarian red (`#c8102e`)
- `--color-accent` — gold (`#f0a500`)
- `--font-body` — Open Sans
- `--font-display` — Annie Use Your Telescope
- `--font-fun` — Love Ya Like A Sister

## Deployment

The `_site/` directory (generated by `jekyll build`) can be deployed to any static hosting provider: GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.

For GitHub Pages, add to `_config.yml`:
```yaml
theme: null
plugins:
  - jekyll-feed
  - jekyll-seo-tag
```
